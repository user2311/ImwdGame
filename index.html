imwd - an interactive web framework

What is it?

imwd is a javascript entity-component-system for game and interactive media development.

imwd provides sane elementary building blocks for multimedia development, and provides workarounds and solutions for many common browser inconsistencies and inconveniences. It is designed to simplify getting a project off the ground while keeping things sustainable for longer development.

Common Issues:

If the game doesn't respond to key presses, try clicking into the game "window" before pressing keys.

If the game doesn't run at all, you likely either need a better browser (try firefox or chrome for best compatibility), or to enable javascript.

No sound on Safari: take it up with Apple, Vorbis is free/open.

How to play the game. The game plays by the use of the following keyboard keys.

  • The web interface starts from the index page where the game initiates and the visualization is formed.
  • After the page s loaded one can use the up,left and right arrow to control the beetle to move arroung.
  • The r key is used to reset the game to an initial phase destroying the current game position.
  • the N button arrow is used when one wants to sho animation.
  • The T button is used to return the game to a starting point. A button is used to change between game options.
  • The "~`" keyboard key inside the quotes is used to show the current mouse position and the FPS (frame per second) of the game.
